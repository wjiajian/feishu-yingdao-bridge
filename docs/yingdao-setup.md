# 影刀端实施手册

本文档只覆盖影刀应用和分享表单配置。

## 一、实施目标

影刀端要完成这些工作：

1. 创建或确认可执行的影刀应用
2. 为每个应用准备分享表单
3. 将分享表单地址提供给飞书配置人员

完成后，每个业务应用至少要提供：

- `app_code`
- `app_name`
- `description`
- `form_url`

## 二、准备影刀应用

每个飞书侧可见应用，影刀侧都应有一个正式可发布的影刀应用版本。

如果应用需要附件上传、复杂校验或多步骤填写，统一放到影刀分享表单里完成，不再依赖飞书卡片表单。

## 三、创建分享表单

1. 打开目标影刀应用
2. 配置表单字段和附件上传能力
3. 发布应用
4. 复制分享表单地址
5. 把地址写入飞书多维表格 `apps.form_url`

![alt text](/resources/影刀高级计划.png)

## 四、交付给飞书侧的字段

| 项 | 说明 |
|---|---|
| `app_code` | 例如 `expense_form` |
| `app_name` | 例如“报销表单” |
| `description` | 飞书卡片说明文案 |
| `form_url` | 影刀分享表单地址 |
| `success_text` | 飞书卡片提示文案，例如“请在影刀表单中提交” |

## 五、影刀端联调

1. 先直接访问分享表单地址，确认页面可打开
2. 手工提交一份测试数据，确认影刀应用能正常触发
3. 再回到飞书里点击应用按钮，确认跳转地址一致

## 六、常见问题

### 1. 分享表单无法访问

重点检查：

- 表单是否已发布
- 分享权限是否正确
- 链接是否复制完整

### 2. 表单可以打开，但提交后业务没执行

重点检查：

- 影刀应用是否已发布最新版本
- 表单提交后的流程配置是否完整
- 运行授权、机器人组是否有效
